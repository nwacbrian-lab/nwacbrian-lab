 # Active Directory Home Lab

## Overview
This project simulates an enterprise Active Directory environment using Windows Server 2022. The lab demonstrates user provisioning, Group Policy configuration, and domain management.

---

## Tools Used
- Windows Server 2022
- Active Directory Domain Services (AD DS)
- DNS
- Group Policy Management
- Windows 10 Client
- VirtualBox

---

## Project Steps

### 1. Domain Controller Setup
- Installed Windows Server 2022
- Configured Active Directory Domain Services
- Promoted server to Domain Controller

### 2. User Creation
- Created organizational units (HR, IT, Sales)
- Created domain users
- Assigned passwords and permissions

### 3. Group Policy Configuration
- Applied password policies
- Configured account lockout settings
- Restricted control panel access

### 4. Client Domain Join
- Joined Windows 10 machine to domain
- Verified login with domain credentials
